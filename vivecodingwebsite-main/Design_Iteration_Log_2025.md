# 2025 디자인 레퍼런스 반영 로그

기준 링크: Webflow "15 best website designs in 2025"  
참고: https://webflow.com/blog/best-websites

## 반영 방식
- 각 사이트에서 1개 핵심 패턴만 추출
- 직접 복제 대신 구조/사용성 원칙으로 변환
- 현재는 데스크톱 기준만 반영(반응형 제외)

## 15개 패턴 매핑
1. SVZ: 강한 히어로 메시지 + 높은 시각 대비 → 메인 히어로 강화
2. Wayside: 내러티브 흐름이 보이는 카드 구성 → "빠른 동선 가이드"
3. VOUS Church: 대형 타이포 중심 정보 우선순위 → 페이지 헤더 계층 정리
4. Ready: 빠른 전환 CTA → 상단 유틸 액션 버튼 강화
5. April Ford: 에디토리얼 카드 질감 → 패널 배경 그라디언트
6. Chiara Luzzana: 감성형 배경 레이어 → 전역 메쉬 배경
7. Pierre-Louis Geldenhuys: 포트폴리오형 카드 리듬 → 목록 카드 밀도 조정
8. Superlist: 생산성 앱식 빠른 이동 UX → 상단 "빠른 이동" 커맨드 추가
9. Michael Kors: 프리미엄 톤의 카드/간격 → 그림자/보더 체계 개선
10. Slite: 문서형 가독성 + 구조적 네비 → 상세 섹션 점프 네비
11. Goonies: 커뮤니티 참여 동선 강화 → 캠프 페이지 팀 탐색/생성 분리
12. Donuts Inc: 밝은 포인트 칩 UI → 상태/마감 칩 체계 강화
13. Digital China University: 정보 블록의 명확한 구획 → 메트릭 카드 도입
14. Monograph Communications: 미니멀 + 강한 정보 계층 → 필터 바 단순화
15. Moon Ultralight: 제품 중심 포커스 카드 → 메인 우측 포커스 패널

## 이번 라운드 결과
- 기술 전환: React + React Router + TypeScript + CSS Modules
- 기능 추가: 상단 빠른 이동 검색, 목록 검색/정렬 고도화, 제출 준비도/히스토리
- 검증: lint/build/typecheck 통과
